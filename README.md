# Invoice generator

## setup
```node
nvm use v22.15.0
```

## run
Run me with:

```npm
npx vite

# or

npm run dev
```

Default port is http://localhost:5173/

## scaffolding

Project was scaffolded with vite:
```npm
npm create vite@latest invoice-generator -- --template react-ts
cd invoice-generator
npm install
npm run dev
```
